# Soldier Project

Instanciation de modèle GLTF/GLB.

## Installation

- Installer les packages :
```bash
npm install
```

- Démarrrer le jeu sur un port de la machine :
```bash
npm run dev
```

- Puis ouvrez un naviateur vers cette adresse : http://localhost:5173/soldier-projet/
